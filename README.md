# Open-Classrooms-Troisième-Projet-OhMyFood

Oh My Food est le troisième projet du parcours de Développeur Web proposé par l'organisme de formation Open Classrooms. Ceci est une version améliorée du projet de base, j'espère que vous apprécierez la visite !

## Contexte :

Vous venez d’être recruté chez Ohmyfood!, en tant que développeur junior. Félicitations !

Ohmyfood! est une jeune startup qui voudrait s'imposer sur le marché de la restauration. L'objectif est de développer un site 100% mobile qui répertorie les menus de restaurants gastronomiques. En plus des systèmes classiques de réservation, les clients pourront composer le menu de leur repas pour que les plats soient prêts à leur arrivée. Finis, les temps d'attente au restaurant !

## Compétences évaluées :

- Utiliser un système de gestion de versions pour le suivi du projet et son hébergement.
- Mettre en œuvre des effets CSS graphiques avancés.
- Mettre en place son environnement Front-End.
- Mettre en place une structure de navigation pour un site web.
- Assurer la cohérence graphique d'un site web.

## Technologies utilisées :

- HTML 5.
- CSS 3.
- SCSS.
